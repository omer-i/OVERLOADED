====================================================
                   OVERLOADED
       A fast-paced multiplayer action game
====================================================

Battle hordes of enemies, compete with other players,
and climb the leaderboard!


----------------------------------------------------
  INSTALLATION
----------------------------------------------------

1. Extract the zip file
2. Double-click  install.bat
3. Click YES on the administrator prompt
4. Wait for the installer to finish (~1 minute)
5. Launch from the desktop shortcut

That's it! The installer handles everything automatically.

If Python installation fails, download and install Python 3
from  https://python.org  then run install.bat again.


----------------------------------------------------
  SYSTEM REQUIREMENTS
----------------------------------------------------

  OS    : Windows 10 or Windows 11
  RAM   : 512 MB minimum
  Other : LAN connection for multiplayer


----------------------------------------------------
  HOW TO PLAY
----------------------------------------------------

Controls:
  WASD          Move
  Mouse         Aim
  Left Click    Shoot
  ESC           Return to menu

Single Player:
  - Pick "Single Player" from the main menu
  - Survive endless enemy waves
  - Enemies get faster the longer you live
  - Your best kill count is saved automatically

Multiplayer (LAN):
  Host  - Log in, click "Create Room", wait for players
  Join  - Log in, click "Join Room", pick a server

Accounts:
  - Register a username and password on first launch
  - Leaderboard shows the top 10 players by best run


----------------------------------------------------
  TROUBLESHOOTING
----------------------------------------------------

Game won't launch:
  Open a terminal in C:\Program Files\Overloaded\
  and run:  python Game.py
  This shows any error messages.

Multiplayer servers not showing up:
  - Make sure all players are on the same network
  - Allow ports 12345 (TCP) and 54321 (UDP) in your firewall
  - Confirm the host is still in the waiting room

Low FPS:
  - Close other applications to free up resources
